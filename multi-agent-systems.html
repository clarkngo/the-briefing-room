Multi-Agent Systems in Smart Manufacturing

This project investigates the decentralized coordination of autonomous agents within a manufacturing environment. By utilizing Multi-Agent Systems (MAS), we can optimize production lines in real-time without central point-of-failure bottlenecks.

Core Objectives

  • Scalable communication protocols between robotic nodes.
  • Dynamic resource allocation based on real-time demand.
  • Self-healing supply chain logistics.

Focus Area: The integration of Large Language Models to orchestrate high-level task planning for low-level mechanical agents.
